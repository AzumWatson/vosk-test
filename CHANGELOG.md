# 更新日志

## v1.1.0 - 2025-11-08

### 🚀 新功能
- ✅ 添加麦克风测试功能 - 点击"🔍 测试麦克风"按钮检测设备
- ✅ 自动音频重采样 - 支持任意采样率自动转换到 16kHz
- ✅ 更友好的错误提示 - 针对不同错误类型提供具体解决方案

### 🔧 优化改进
- 移除强制 16kHz 采样率限制，使用设备默认采样率
- 优化音频参数设置（回声消除、噪声抑制、自动增益）
- 移除音频输出连接，避免回声问题
- 改进麦克风权限检测逻辑

### 📝 文档更新
- 新增 `MICROPHONE_TROUBLESHOOTING.md` - 详细的麦克风故障排查指南
- 新增 `QUICK_START.md` - 快速开始指南
- 更新 `README.md` - 添加常见问题解答
- 更新界面提示信息

### 🐛 Bug 修复
- ✅ 修复 "Requested device not found" 错误
- ✅ 修复采样率不兼容导致的设备无法访问问题
- ✅ 修复音频处理器重采样逻辑

## v1.0.0 - 2025-11-08

### 🎉 首次发布

#### ✨ 核心功能
- 完全离线的语音识别应用
- 基于 SvelteKit + Vosk-Browser
- 支持中文语音识别（Vosk 小型模型）
- 实时转写和最终结果显示
- 本地模型加载，避免 CORS 问题

#### 🎨 用户界面
- 现代化渐变色背景设计
- 响应式布局
- 实时录音状态显示
- 清空文本功能

#### 🔒 隐私安全
- 所有处理在浏览器本地完成
- 数据不离开浏览器
- 模型文件本地存储

#### 📦 技术栈
- SvelteKit - Web 框架
- Vosk-Browser - WebAssembly 语音识别引擎
- Web Audio API - 音频处理
- AudioWorklet - 高性能音频流处理

#### 📁 项目结构
- `src/routes/+page.svelte` - 主页面
- `static/audio-processor.js` - 音频处理器
- `static/models/` - 模型文件目录
- `download-model.sh` - 自动下载脚本
